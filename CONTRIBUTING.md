# Contributing to TapMorse

Thanks for your interest in improving TapMorse! Contributions are welcome.

## Getting Started
- Python 3.10+ recommended (Windows supported)
- Install deps: `python -m pip install -r requirements.txt`
- Run: `python table_click_detector.py`
- Practice tool: `python morse_practice.py`
- Diagnostics: `python diagnostics.py`

## Ways to Contribute
- Bug reports and reproducible issues
- Feature requests (open an issue to discuss)
- Pull requests (small, focused changes preferred)
- Documentation and examples

## Development Guidelines
- Keep changes minimal and focused; avoid unrelated refactors
- Preserve existing public behavior unless the change is approved
- Follow current code style and naming patterns
- Include a short description in PRs: problem, approach, testing

## Branching & PRs
- Create a feature branch from `main`
- Ensure the app runs locally (no crashes)
- Add notes for any user-facing changes (README updates welcome)

## Reporting Issues
Include:
- OS and Python version
- Steps to reproduce (commands, inputs)
- Expected vs actual behavior
- Logs/traces if available

## Community Conduct
Be respectful and constructive. Collaborate and help newcomers.
